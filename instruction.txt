1. To run the application, follow the instructions in the `Readme.md` file.
2. After launching the application in the console, you will see all the project entries by ID,
which you indicated by executing the instruction in the `Readme.md` file.
3. A report `report.xlsx` file will be created. It has 2 worksheets:
the first one for complete information about tasks, and the second for tasks
and durations grouped by start dates.
